Examples:
```ruby

balanced?('What is) this?') == false
balanced?('What (is this?') == false

balanced?('((What)) (is this))?') == false
balanced?('Hey!') == true
balanced?(')Hey!(') == false
balanced?('What ((is))) up(') == false
Note that balanced pairs must each start with a (, not a ).
